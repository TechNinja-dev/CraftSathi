## 📖 About The Project

**CraftSathi** is a comprehensive, modern digital ecosystem empowering local artisans to achieve global identity and exposure. We equip traditional craftspeople with cutting-edge AI tools (for automated image, video, and storytelling generation), a dedicated community social network, and intelligent strategic guidance to seamlessly navigate modern e-commerce.

## ✨ Key Features & Workflow

- **🧠 Smart AI Engine:** Automated generation of high-converting social media posts, engaging stories, and smart product tags using advanced language models. Artisans can generate assets effortlessly.
- **🌐 Artisan Community Hub:** A dedicated social media network allowing craftspeople globally to connect, share their portfolios, and scale their businesses together.
- **🧭 Strategic Craft Guidance:** Actionable market analytics determining a craft's local and global market worth alongside tailored selling routes and pricing models.
- **🪪 Identity Builder:** Creation of secure, verifiable digital craft passports to prove product authenticity and trace historical lineage.
- **📈 Live Analytics Tracking:** An auto-scaling internal backend infrastructure driving active metrics for total supported users, generation counts, and participating countries natively into the UI.

## 🛠️ The Tech Stack

### Frontend Architecture
- **Framework:** React.js (Vite)
- **Styling:** Custom Vanilla CSS & Tailwind CSS (Dark-themed glassmorphism UI)
- **Animations:** Framer Motion (Iterative timeline & scroll-spy rendering)
- **Routing:** React Router DOM v6
- **Data Vis:** Recharts & CountUp algorithms

### Backend Architecture
- **Framework:** FastAPI (Python API router)
- **Database:** MongoDB (PyMongo connection scaling)
- **Security & Auth:** Firebase Admin SDK & custom OTP-based verifications
- **AI Integrations:** Bespoke python automation handling multimedia (images, videos, tags)

## 🚀 Installation & Setup

Follow these steps to establish the environment on your local system:

### Prerequisites
- [Node.js](https://nodejs.org/en/) (v16+)
- [Python](https://www.python.org/) (3.9+)
- Active MongoDB cluster/connection string
- Firebase Service Account Key JSON

### 1. Clone the repository
```bash
git clone https://github.com/your-username/CraftSathi.git
cd CraftSathi
```

### 2. Frontend Execution
```bash
cd frontend
# Install node modules
npm install
# Spin up development server
npm start
```
*The frontend should now be running on `http://localhost:3000` or `5173`.*

### 3. Backend Execution
Open a second terminal window:
```bash
cd backend
# Install dependencies
pip install -r requirements.txt

# Run the FastAPI server in reload mode
uvicorn main:app --reload
```
*The backend API should now be running locally on `http://127.0.0.1:8000`.*
